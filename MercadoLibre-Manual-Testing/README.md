# \#\\# Proyecto de Automatización - Urban Routes

# 

# \# \\## Descripción



# \#Este proyecto contiene pruebas automatizadas para la aplicación web Urban Routes, 
# una plataforma de solicitud de taxis. Las pruebas verifican el flujo completo 
# desde la configuración de ruta hasta la solicitud del taxi.

# 

# ---

# 

# \#\\## Funcionalidades Automatizadas

# 

# Las pruebas cubren el siguiente flujo completo:

# 

# 1\. Configuración de dirección (origen y destino)

# 2\. Selección de la tarifa \*\*Comfort\*\*

# 3\. Ingreso de número de teléfono y verificación SMS

# 4\. Configuración de Método de pago (registro de tarjeta de crédito)

# 5\. Escritura de mensaje para el conductor

# 6\. Solicitud de manta y pañuelos

# 7\. Selección de servicios adicionales (Manta, pañuelos y 2 helados)

# 8\. Solicitud final del taxi

# 9\. Verificación del modal de búsqueda del conductor (opcional)

# 

# ---

# 

# \#\\## Tecnologías Utilizadas

# 

# \- \*\*Python 3\*\* -Lenguaje de programación principal.

# \- \*\*Selenium WebDriver\*\* Framework de automatización web

# \- \*\*PyTest\*\* Framework de testing

# \- \*\*WebDriverWait\*\* - Para esperas explícitas



# \- \*\*ChromeDriver\*\* -Navegador para ejecutar las pruebas

# \- \*\*PyCharm\*\*

# \- \*\*Page Object Model (POM)\*\*- Patrón de diseño para organizar el código
